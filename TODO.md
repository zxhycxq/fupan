# 考试成绩分析系统开发计划

## 项目概述
开发一个考试成绩分析系统,通过上传考试截图,利用OCR技术提取成绩数据,进行多维度分析和可视化展示。

## 技术栈
- 前端：React + TypeScript + Tailwind CSS + shadcn/ui
- 图表：ECharts for React
- 后端：Supabase（数据存储）
- OCR：图像内容理解API
