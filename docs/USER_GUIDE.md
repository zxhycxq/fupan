# 考试成绩分析系统使用指南

## 系统简介

考试成绩分析系统是一个专门用于分析和管理模拟考试成绩的Web应用。通过上传考试截图,系统会自动识别图片中的成绩数据,并进行多维度分析和可视化展示,帮助您了解学习状况,制定针对性提升计划。

## 主要功能

### 1. 数据分析仪表板

仪表板提供了全面的成绩分析视图:

- **统计卡片**: 显示考试次数、平均分、最高分和平均用时
- **总分趋势图**: 折线图展示历次考试的总分变化趋势
- **各模块平均正确率**: 柱状图对比各大模块的平均正确率
- **得分分布**: 饼图展示不同分数段的考试次数分布

### 2. 上传考试成绩

上传功能让您轻松添加新的考试记录:

1. 输入考试期数(如第1期、第2期等)
2. 选择考试成绩截图(支持JPG、PNG等格式,每张最大10MB)
   - **支持一次上传多张图片**: 如果您的成绩分布在多张截图中,可以一次性选择所有图片
   - 系统会依次识别每张图片,然后合并所有数据
3. 预览已选择的图片,可以单独删除不需要的图片
4. 点击"上传并解析"按钮
5. 系统会自动识别所有图片中的成绩数据并保存

**注意事项**:
- 确保所有截图清晰,包含完整的成绩信息
- 图片中应包含总分、用时、各模块得分等数据
- 如果成绩信息分散在多张图片中,请一次性上传所有相关图片
- 系统会自动合并多张图片的识别结果
- 每张图片最大10MB,建议上传清晰度适中的截图

### 3. 考试记录管理

考试记录页面列出了所有历史考试:

- 查看每次考试的期数、总分、用时、平均分和击败率
- 点击"眼睛"图标查看详细信息
- 点击"删除"图标删除不需要的记录

### 4. 考试详情分析

详情页面提供深入的单次考试分析:

- **基本信息**: 总分、用时、平均分、击败率
- **弱势模块提醒**: 自动识别正确率低于60%的模块
- **各模块正确率雷达图**: 直观展示各模块的正确率分布
- **各模块用时对比**: 柱状图显示各模块的答题用时
- **模块详细数据**: 展示每个大模块和子模块的详细答题情况

## 使用建议

1. **定期上传**: 每次模拟考试后及时上传成绩,建立完整的学习档案
2. **关注趋势**: 通过总分趋势图观察自己的进步情况
3. **针对性提升**: 重点关注弱势模块,制定专项提升计划
4. **时间管理**: 分析各模块用时,优化答题时间分配
5. **对比分析**: 对比不同时期的成绩,评估学习效果

## 常见问题

### Q: 图片识别不准确怎么办?
A: 请确保上传的截图清晰完整,包含所有必要的成绩信息。如果识别结果不理想,可以删除记录后重新上传更清晰的截图。

### Q: 数据会丢失吗?
A: 所有数据都保存在云端数据库中,不会因为清除浏览器缓存而丢失。

### Q: 支持哪些考试类型?
A: 系统主要针对公务员行测考试设计,但可以适用于其他具有类似模块结构的考试。
